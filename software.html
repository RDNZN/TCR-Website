Beschikbaarheid over de software

Bij deze opleiding ga je veel gebruik maken van verschillende software. Een paar voorbeelden zijn: HTML, CSS, het office pakket, SQL, etc. Alle software die je nodig hebt krijg je van school doorgestuurd. Je moet veel software gebruiken voor presentaties, opslag van je documenten, huiswerk en leerwerk. Als je software nodig hebt die je nog niet tot je beschikking hebt kan de school het bijna altijd voor je regelen want sommige software word niet meer uitgegeven zoals Photoshop maar vaak is er wel een alternatief dat je kan gebruiken. Het word aangeraden om de software te gebruik die je van school heb gekregen maar je mag zelf beslissen wat jij wilt gebruiken. Als jij toch een andere software gebruikt omdat jij het persoonlijk fijner vind dan kan je dat natuurlijk ook gebruiken.

Wat is iedere software precies

HTML

HTML is een software waarmee je de basis van je website maakt. Als je text, foto's, videos en nog veel meer erin zou willen hebben maak je dat met HTML.

CSS

CSS is een programma dat je gebruikt om de style van je website te veranderen. Als je achtergronden een speciale navigatie bar of een copyright wilt hebben dan kun je CSS gebruiken om het allemaal te veranderen zodat het eruit ziet zoals jij het wilt hebben.

Office

Office is een programma dat iedereen wel eens heeft gebruikt. Het office pakket bevat: Word, Excel, Powerpoint, OneNote, Outlook, Publisher en Acces. Deze programma's gebruikt je voornamelijk voor huiswerk of presentaties maken.

MySQL

MySQL is een programma voor databases. Als je simpel iets wilt zoeken in een database of als je een database wilt aanpassen kan het altijd met MySQL.

Notepad ++

Notepad ++ is een programma waarin je vaak je HTML files in verwerkt. Je kan ook altijd alternatieve opties gebruiken want zelf vinden wij dat Atom beter werkt dan Notepadd ++ maar het is wel het simpelste om te gebruiken
