Beschikbaarheid over de software

Bij deze opleiding ga je veel gebruik maken van verschillende software. Een paar voorbeelden zijn: HTML, CSS, het office pakket, SQL, etc.
Alle software die je nodig hebt krijg je van school doorgestuurd. Je moet veel software gebruiken voor presentaties, opslag van je documenten, huiswerk en leerwerk.
Als je software nodig hebt die je nog niet tot je beschikking hebt kan de school het bijna altijd voor je regelen want sommige software word niet meer uitgegeven zoals Photoshop
maar vaak is er wel een alternatief dat je kan gebruiken. Het word aangeraden om de software te gebruik die je van school heb gekregen maar je mag zelf beslissen wat jij wilt gebruiken.
Als jij toch een andere software gebruikt omdat jij het persoonlijk fijner vind dan kan je dat natuurlijk ook gebruiken.

USB Webserver

Dit wordt gebruikt om de MySQL-Databases te beheren.

Office

Office is een programma dat iedereen wel eens heeft gebruikt. Het office pakket bevat: Word, Excel, Powerpoint, OneNote, Outlook, Publisher en Acces. Deze programma's gebruikt je voornamelijk voor huiswerk of presentaties maken.

Datamodeler

Dit is een pakket om MySQL-Databases te modeleren.

Notepad ++

Notepad ++ is een programma waarin je vaak je HTML files in verwerkt. Je kan ook altijd alternatieve opties gebruiken want zelf vinden wij dat Atom beter werkt dan Notepadd ++ maar het is wel het simpelste om te gebruiken
